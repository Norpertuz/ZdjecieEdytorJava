Dołączam pliki .jar, które w razie błędów mówiących o brakujących ścieżkach bądź gdy nie działa histogram, trzeba dodać do projektu:
W Eclipse: Prawy przycisk na projekt -> Build Path -> Add External Archives -> Zaznaczyć trzy pliki .jar dodane przeze mnie.
W razie jakby po dodaniu plików dalej występował błąd brakujących ścieżek, należy go zignorować - program i tak się uruchomi, a histogram już będzie działał.
